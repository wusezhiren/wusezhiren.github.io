## 一、核心机制改造(最高优先 —— 决定"像不像 DNF")  ✅ 已全部完成

### 1. 地下城房间制(替换现有"波次自动刷怪")  ✅
- **房间结构**:一张地下城 = 若干房间串联(例:5~6 个普通房 + 1 个 Boss 房)。
- **进房刷怪一次**:进入房间时一次性生成该房怪物,**不再自动刷新下一波**。
- **打完开门**:房内怪物全部清空后,**屏幕右侧(或对应方向)出现"门"**并播放开门特效/提示箭头。
- **走进去切换**:玩家**走到门边触碰**才进入下一个房间(带短暂切场过渡),而非自动跳波。
- **关底 Boss 房**:最后一个房间是 Boss;击杀后该地下城**通关结算**(掉落 + 经验 + 金币 + 评分)。
- **保留纵深与多门**:可选支持上/下/右多个门(先做"右侧单门"最小可用,后续扩展分支)。
- 状态机改造:`gameState` 增加 `roomClear`(待开门)、`transition`(切场);`wave` 概念替换为 `dungeon`(第几图)+ `room`(第几房)。

### 2. 装备系统(武器 / 防具)  ✅
- **装备槽**:武器、头、上衣、下装、鞋(可先做"武器 + 防具×2"最小集,再扩)。
- **属性加成**:装备提供 攻击 / 最大HP / 最大MP / 暴击率 / 攻速 等加成,叠加到现有 `dmgMul()` 等计算。
- **稀有度**:普通(白)/ 高级(蓝)/ 稀有(紫)/ 史诗(橙),颜色 + 词条数量区分。
- **Boss 概率掉落装备**:击杀 Boss / 精英怪按概率掉落装备(地面发光掉落物,拾取进背包)。
- **职业限定**:鬼剑士系武器(太刀/巨剑/钝器等)外观与微属性差异(先做数值,后做外观)。

### 3. 背包系统(I 键打开)  ✅
- **按 I 打开/关闭背包**,**不暂停游戏**(怪照打、战斗继续,背包作为半透明叠加层),需注意操作时仍可被攻击。
- **网格物品栏**:显示装备与消耗品,鼠标/方向键选中。
- **操作**:穿戴/卸下装备、查看词条对比(↑绿↓红)、丢弃、把消耗品**指派到 1~6 快捷栏**。
- **角色面板**:显示当前装备槽与人物总属性(攻/防/HP/MP/暴击)。

### 4. 消耗品快捷栏(数字键 1~6)  ✅
- 屏幕底部(技能栏旁/上方)显示 **6 个消耗品格**。
- **按 1~6 立即使用**对应格子的消耗品(HP药 / MP药 / 增益药等),走位中可用、有数量与短 CD。
- 消耗品来源:怪物掉落、商店购买;用尽后格子置灰。

### 5. NPC 商店(随进随出,无时间限制)  ✅
- **取消"每3波强制局间商店"**;改为可随时进入的商店(地下城入口/城镇 NPC,或按键呼出)。
- **出售三类**:
  - **武器 / 防具**(按当前进度刷新可购列表)
  - **药品 / 消耗品**(HP药、MP药、增益药,可补充进背包/快捷栏)
  - (保留)属性强化作为可选项或迁移为装备
- **买卖双向**:可**卖出**背包里多余装备换金币。
- 无倒计时、无强制,逛完按键离开继续刷图。
